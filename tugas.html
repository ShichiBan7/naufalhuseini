10 Contoh Jenis Font Populer

1. Roboto

Ini adalah contoh teks menggunakan font Roboto. Font sans-serif modern yang sangat populer untuk desain web.

2. Pacifico

Ini adalah contoh teks menggunakan font Pacifico. Font script santai yang terinspirasi oleh tulisan tangan.

3. Dancing Script

Ini adalah contoh teks menggunakan font Dancing Script. Font kaligrafi yang elegan dan mengalir.

4. Montserrat (Bold)

Ini adalah contoh teks menggunakan font Montserrat dengan ketebalan bold. Font geometris yang bersih dan modern.

5. Indie Flower

Ini adalah contoh teks menggunakan font Indie Flower. Font tulisan tangan yang terlihat alami dan tidak sempurna.

6. Courier Prime

Ini adalah contoh teks menggunakan font Courier Prime. Font monospace klasik yang sering digunakan untuk kode.

7. Abril Fatface

Ini adalah contoh teks menggunakan font Abril Fatface. Font display dengan kontras tinggi yang elegan.

8. Playfair Display (Bold)

Ini adalah contoh teks menggunakan font Playfair Display dengan ketebalan bold. Font serif tinggi yang klasik.

9. Comfortaa

Ini adalah contoh teks menggunakan font Comfortaa. Font sans-serif bulat yang modern dan ramah.

10. Georgia (Italic)

Ini adalah contoh teks menggunakan font Georgia dengan gaya italic. Font serif yang dirancang khusus untuk keterbacaan di layar.

10 Contoh Format File Gambar Populer

1. JPEG (.jpg/.jpeg)
Format lossy yang bagus untuk foto dengan ukuran file kecil
2. PNG (.png)
Format lossless dengan dukungan transparansi
3. GIF (.gif)
Format untuk gambar animasi dan warna terbatas (256)
4. SVG (.svg)
Gambar vektor yang bisa di-scaling tanpa kehilangan kualitas
5. WebP (.webp)
Format modern dengan kompresi lebih baik dari JPEG/PNG
6. AVIF (.avif)
Format gambar baru berbasis AV1 dengan kompresi efisien
7. BMP (.bmp)
Format gambar bitmap tanpa kompresi (ukuran file besar)
8. ODD (.odd)
Format gambar OpenDocument Drawing
9. ICO (.ico)
Format khusus untuk ikon/favicon
10. HEIF/HEIC (.heif/.heic)
Format gambar modern dengan kompresi efisien (digunakan iOS)

10 Contoh Format Audio Populer

1. MP3 (.mp3)
Format kompresi audio paling populer dengan kualitas baik dan ukuran kecil
2. WAV (.wav)
Format audio tanpa kompresi, kualitas tinggi tetapi ukuran file besar
3. OGG Vorbis (.ogg)
Format open source dengan kompresi lebih baik dari MP3
4. AAC (.aac)
Format standar untuk YouTube, iPhone dan iTunes
5. FLAC (.flac)
Format lossless open source dengan kompresi tanpa kehilangan kualitas
6. Opus (.opus)
Format sangat efisien untuk streaming dan komunikasi real-time
7. WebM Audio (.weba)
Format audio dari proyek WebM, biasanya berisi kodek Opus atau Vorbis
8. M4A (.m4a)
Format kontainer Apple untuk audio AAC, umum di iTunes
9. MIDI (.mid)
Format data musik bukan audio sebenarnya, berisi instruksi untuk synthesizer
10. AMR (.amr)
Format untuk rekaman suara di ponsel, kompresi sangat tinggi

10 Contoh Format Video Populer

1. MP4 (.mp4)
Format paling kompatibel di web. Menggunakan codec H.264 untuk video dan AAC untuk audio. Dukungan Tinggi
2. WebM (.webm)
Format open source dari Google. Menggunakan codec VP9 untuk video dan Opus untuk audio. Dukungan Tinggi
3. OGG Theora (.ogg atau ogv)
Format open source alternatif. Menggunakan codec Theora untuk video dan Vorbis untuk audio. Dukungan Sedang
4. MP4 (H.265/HEVC) (.mp4)
Generasi berikutnya dari H.264 dengan kompresi lebih efisien. Dukungan browser masih terbatas. Dukungan Sedang
5. WebM (AV1) (.webm)
Format next-gen dengan codec AV1 yang sangat efisien. Dukungan masih berkembang. Dukungan Sedang
6. QuickTime (.mov)
Format kontainer Apple. Biasanya berisi codec H.264. Di web, lebih baik konversi ke MP4. Dukungan Minimal
7. AVI (.avi)
Format kontainer lama dari Microsoft. Dukungan browser sangat terbatas. Dukungan Minimal
8. MKV (.mkv)
Format kontainer fleksibel tetapi dukungan native di browser sangat terbatas. Dukungan Minimal
9. 3GP (.3gp)
Format untuk perangkat mobile. Biasanya berisi codec H.263 atau MPEG-4 Part 2. Dukungan Minimal
10. FLV (.flv)
Format Flash Video yang sudah usang. Tidak didukung secara native di browser modern. Dukungan Minimal
